# Final Demo Script

## Opening Pitch

AutoOps is a local-first, production-style DevOps Control Plane for governed CI/CD, containers, Kubernetes, infrastructure automation, observability, and cloud-readiness evaluation.

## 7-Minute Demo

1. Login as Operator / Requester.
2. Open Operations Hub and show runtime health.
3. Open Jenkins, Docker, Kubernetes, and Infrastructure Automation.
4. Show Terraform/OpenTofu and Ansible safe local execution.
5. Open Governance Center and show evidence.
6. Open GitHub Actions, Observability, DevOps Tools, and Cloud Readiness.
7. Show incidents/runbooks and final release docs.

## 15-Minute Technical Demo

Add:

- Approval-gated Docker restart or Terraform apply request.
- Requester self-approval failure.
- Admin approval path.
- Operation detail lifecycle.
- CI/release gate and secret scan.
- Final smoke-check script.

## Closing Pitch

AutoOps is not claiming enterprise certification. It shows production-grade architecture principles, real integrations, safe governance, and a clear path to company pilot hardening.
