# Final Screenshot Checklist

Recommended order:

1. Dashboard overview.
2. Operations Hub.
3. Governance Center.
4. Jenkins connector.
5. GitHub Actions connector.
6. Docker connector.
7. Kubernetes connector.
8. Infrastructure Automation Center.
9. Observability Integrations.
10. DevOps Tools Readiness.
11. Cloud Provider Readiness.
12. Incident runbook.
13. CI workflow green.

Blur or crop:

- Browser tabs and personal profile details.
- Emails if preferred.
- Operation UUIDs if preferred.
- Internal hostnames.
- Error text that may include environment-specific paths.

Never upload:

- `.env`
- tokens
- kubeconfig
- private keys
- Terraform state
- Ansible vault files
- Authorization headers
- database credentials
- raw logs with secrets
